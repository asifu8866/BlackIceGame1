# Black Ice Gaming - Company Context

## Team
- **Ike** (founder) - 18 years gaming industry experience, directs vision and game design, no coding background

## Tools and Stack
- Single HTML file architecture (game.html)
- HTML5 Canvas for graphics
- JavaScript for game logic
- Chrome browser in kiosk mode for runtime
- Git for version control

## Business Model
- Proprietary skill game software for COAM cabinets
- Hardware cost: $1,400 per unit (vs $6,500 competitor)
- Target: Georgia COAM market first, then multi-state
- 100% offline operation - no internet at locations
