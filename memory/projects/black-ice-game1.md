# Black Ice Gaming - Game 1

## Overview
First game title for Black Ice Gaming's COAM skill game cabinet. Single HTML file running offline on Intel i5 / Windows 10 IoT hardware with JAMMA 36-pin arcade cabinet.

## Key Specs
- 5 reels, 3 rows, 10 paylines
- 32% hit frequency, medium volatility
- RTP: 88%, 92%, or 96% (operator-selectable)
- Hold and Spin bonus every 80-120 spins
- 4 jackpot levels: Mini, Minor, Major, Mega
- Skill elements: Nudge + Symbol Switch
- Target hardware: JAMMA cabinet, Intel i5, Windows 10 IoT
- Cost per unit: $1,400

## Market
- Primary: Georgia COAM market
- Expansion: multi-state deployment
- Competitor: Banilla Fusion at $6,500/unit
