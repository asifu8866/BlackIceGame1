# Black Ice Gaming - Glossary

## Industry Terms
| Term | Meaning |
|------|---------|
| **COAM** | Coin Operated Amusement Machine - Georgia's legal category for skill games |
| **TITO** | Ticket In Ticket Out - players insert/cash out with printed tickets |
| **SAS** | Slot Accounting System - protocol for machine-to-back-office reporting |
| **RTP** | Return To Player - percentage the game pays back over time (we target 88/92/96%) |
| **JAMMA** | Japan Amusement Machine and Marketing Association - 36-pin wiring standard for arcade cabinets |
| **GLI** | Gaming Laboratories International - the certification company that approves gaming devices |
| **Hit frequency** | How often a spin wins something (we target 32%) |
| **Hold** | Percentage the machine keeps (100% minus RTP) |
| **Volatility** | How spread out wins are - medium means mix of small and big wins |

## Technical Terms
| Term | Meaning |
|------|---------|
| **rAF** | requestAnimationFrame - the fast animation method in JavaScript |
| **Canvas** | The HTML5 drawing area where game graphics render |
| **Kiosk mode** | Fullscreen browser with no toolbar or buttons visible |

## Game-Specific Terms
| Term | Meaning |
|------|---------|
| **Hold and Spin** | Bonus round where Fireballs lock on reels and player gets 3 respins |
| **Nudge** | Skill feature - player shifts a reel one position up or down |
| **Symbol Switch** | Skill feature - player swaps one symbol for another |
| **Fireball** | Special symbol that triggers and locks during Hold and Spin bonus |
